The Problem

AI assistants forget everything

Every conversation starts from zero. Your agent doesn't know what it learned yesterday, can't track how facts change over time, and has no way to connect related information across sessions. Context windows are a cache, not a memory.

How It Works

From text to temporal knowledge graph

Every piece of text your agent ingests becomes structured, searchable knowledge with a timeline.

Features

What makes it different

🕰️

Temporal Knowledge Graph

Every entity and relation carries valid_from / valid_until timestamps. Point-in-time snapshots let you query the graph at any moment.

🔎

Hybrid Search + RRF

HNSW vector similarity and BM25 keyword search, fused with Reciprocal Rank Fusion (K=60). LLM-powered query expansion.

🔗

MCP Native

6 tools, browsable entity resources, and 3 prompt templates. Works with Claude Desktop, Cursor, and any MCP-compatible client.

⚙️

Trait-Based Architecture

Core defines pure traits for embedders, extractors, and query rewriters. Swap providers without touching the pipeline.

💾

SurrealDB All-in-One

One database for documents, graph edges, vector indexes, and full-text search. No middleware glue.

🐳

Ship Anywhere

Run as an MCP server (stdio or HTTP), a CLI tool, or a Docker container. RocksDB persistence by default.

MCP Interface

10 tools your agent can call

add_memory

Ingest text, extract entities and relations, store everything with embeddings. Returns a diff of what changed.

search_memory

Hybrid vector + keyword search with RRF fusion. Filter by entity type.

expand_search

LLM rewrites your query into semantic variants, searches each, and merges results.

get_entity

Look up any entity by name. Get its type, summary, temporal bounds, and relationships.

snapshot

Point-in-time graph state. Pass a timestamp, get back every entity and relation.

list_recent

The N most recent memories, ordered by creation time.

list_agents

See which AI agents have contributed to the graph, with namespaces and episode counts.

list_namespaces

List all namespaces in the graph with entity counts for multi-tenant visibility.

agent_activity

Audit a specific agent's recent contributions by agent_id.

cross_namespace_search

Search across all namespaces globally, ignoring namespace scoping.

Quick Start

Three commands to memory

# Install the CLI
brew install mindbend0x/context-keeper/context-keeper

# Or build the MCP server from source
git clone https://github.com/mindbend0x/context-keeper.git && cd context-keeper
cargo build --release -p context-keeper-mcp

# Add to your MCP client config (Claude Desktop, Cursor, etc.)
{
  "mcpServers": {
    "context-keeper": {
      "command": "context-keeper-mcp"
    }
  }
}

# Or test with the CLI
context-keeper add --text "Alice is a senior engineer at Acme Corp"
context-keeper search --query "Who works at Acme?"
